# CognitoScope – Agentic AI Project

## Overview
CognitoScope is an agentic AI system that focuses on analyzing reasoning patterns instead of only checking final answers.

The system interacts with the user through a question–response loop and evaluates clarity, logical structure, and depth of explanation.

## Objective
To build an AI assistant that:
- Analyzes reasoning quality
- Generates adaptive follow-up questions
- Tracks reasoning improvement over multiple interactions
- Produces a basic cognitive insight report

## Planned Features
- Interactive question-response interface
- LLM-based reasoning analysis
- Adaptive follow-up question generation
- Basic scoring system (clarity, depth, logic)
- Response tracking using SQLite

## Tech Stack
- Python
- Streamlit
- LLM API (Gemini/OpenAI)
- SQLite (for storing interactions)
